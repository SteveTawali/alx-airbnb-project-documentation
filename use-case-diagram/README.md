# Use Case Diagram

This diagram illustrates the core interactions between users (Guest, Host, Admin) and the system in the Airbnb Clone backend.

## Actors and Use Cases

### Guest
- Register / Login
- Search properties
- Book a property
- Cancel booking
- Make payment
- Leave a review
- Send message

### Host
- Register / Login
- Add, Edit, or Delete a property
- View bookings
- Respond to messages

### Admin
- View all users
- Remove user or listing
- Access reports
